## Sorting, Filtering and Grouping in Pivot Tables

### Sorting in Pivot Tables
Sorting helps arrange data in a specific order.

You can:

- Sort values from **Largest to Smallest**
- Sort values from **Smallest to Largest**
- Sort alphabetically (A–Z or Z–A)

**Steps:**
1. Right-click on a value in the Pivot Table.
2. Select **Sort**.

### Filtering in Pivot Tables
Filtering allows learners to display only the data they want to analyze.

You can:

- Filter by specific values
- Using **Value Filters**
- Apply **Top 10 Filter**
- Select specific categories

**Steps:**
1. Click the drop-down arrow in the Pivot Table field.
2. Choose **Filter**.
3. Select the required filtering option.

## 3. Grouping in Pivot Tables

Grouping helps organize data into meaningful categories.

You can group:

- Dates - Months, Quarters, Years
- Numbers - Ranges
- Text - Custom groups
