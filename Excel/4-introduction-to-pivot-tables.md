## Pivot Tables in Microsoft Excel

A `Pivot Table` is a tool used to summarize, analyze, organize, and present large amounts of data quickly.

It allows you to:

- Calculate totals
- Count data
- Find averages
- Compare categories
- Create reports without complex formulas

### Why Use Pivot Tables?

Instead of writing formulas like:

- SUMIFS
- COUNTIFS
- AVERAGEIFS

You can drag and drop fields to:

- Summarize data instantly
- Change reports dynamically
- Create professional dashboards
- Analyze large datasets easily

### Requirements Before Creating a Pivot Table
Your data must:

- Be in tabular format
- Have headers (column names)
- Have no completely blank rows
- Have consistent data types in columns

### How to Create a Pivot Table

1. Select any cell inside the dataset.
2. Insert - PivotTable
3. Choose:
- Table/Range
- New Worksheet (recommended)
4. Click OK.
